# StoreKit Core — Frontend

Frontend do StoreKit Core desenvolvido com React, Vite e Tailwind CSS.

---

# Stack

- React
- Vite
- Tailwind CSS
- React Router
- Context API
- Axios

---

# Funcionalidades

- Home comercial
- Catálogo de produtos
- Página individual
- Variantes de produto
- Carrinho multi-produto
- Persistência local
- Checkout via WhatsApp
- Responsividade refinada
- White-label básico

---

# Estrutura

```txt
src/
├── components/
├── config/
├── features/
├── layouts/
├── styles/
└── utils/
```

---

# Configuração

O frontend utiliza configs desacopladas para:
- branding
- tema
- home
- integração futura com ecossistema

---

# Executando

```bash
npm install
npm run dev
```

---

# Objetivo

Servir como base reutilizável para múltiplos projetos de e-commerce.
